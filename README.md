# 🛡️ FakeCheck AI
### Multimodal Misinformation Detection System

FakeCheck AI is an AI-powered platform designed to detect misinformation across multiple media formats including **text, images, and videos**. With the rise of generative AI and deepfake technologies, it has become increasingly difficult to distinguish between authentic and manipulated content.

This project provides a **multimodal detection system** that analyzes different types of content and predicts whether the content may be fake or manipulated.

---

## 🚀 Features

- 📰 Fake news detection from text
- 🖼️ Deepfake detection for images
- 🎥 Manipulated video detection
- 📊 Probability score for authenticity
- 🌐 Interactive web interface built with Streamlit

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/velloredeepika9-cell/FakeCheck-AI.git
```

### 2. Navigate to the project directory

```bash
cd FakeCheck-AI
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

---

## 🖥️ Usage

1. Launch the Streamlit application.
2. Upload an **image**, **video**, or enter **text content**.
3. The system analyzes the content.
4. A **probability score** will indicate whether the content may be fake or manipulated.
